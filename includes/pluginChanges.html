0.2.0

  • Added the SkillsJars discovery pipeline: Maven dependency scanner, Maven plugin dependency scanner, JAR-level SKILL.md parser, and a project-level SkillRegistry service.
  • Introduced the dev.dong4j.idea.skillsjars.helper.skillSourceScanner extension point so future Gradle/SBT scanners and third-party plugins can plug in without changes to the coordinator.
  • Added the Agent Skills tool window with a sortable table (Skill / Description / Source / Artifact / Allowed Tools) and Refresh / Open Source Jar actions.
  • Added unit tests for frontmatter parsing, JAR parsing, coordinate parsing, and the table model.

0.1.0

  • Initialized the JetBrains IDE plugin project from the non-AI template.
  • Added the base Gradle IntelliJ Platform project structure.
  • Added the initial editor action entry for future SkillsJars refresh and discovery workflows.
  • Added plugin metadata, bundle resources, icons, and notification wiring.
