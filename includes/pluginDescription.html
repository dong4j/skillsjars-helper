SkillsJars Helper

SkillsJars Helper brings SkillsJars dependency visibility into the IDE. It helps Java projects find skills from Maven or Gradle dependencies, preview SKILL.md directly from JAR files, and export selected skills to local Agent directories.

Core Capabilities

  • Dependency scanning: discovers skills under META-INF/skills/**/SKILL.md.
  • SkillsJar preview: opens skill frontmatter and Markdown content without extracting first.
  • Agent export: prepares extraction to directories such as .claude/skills, .codex/skills, and .junie/skills.
  • Marketplace workflow: provides a foundation for SkillsJars search and dependency installation.
  • Publishing preparation: validates local skills before packaging and publishing.

Development Status

The project is currently in the initialization phase. The first implementation milestone is dependency scanning and local preview for SkillsJars content.
